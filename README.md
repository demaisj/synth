# synth
*experiment: \~modular audio synthetiser thingy\~*

## what

### I/O:
- MIDI input
- Pulseaudio output
 
### Voices:
- (Sine|Saw|Tri|Square) oscillator
- 16 oscillators unison
- Envelope
 
### Effects:
- (High|Low|Band)-pass -24dB filter
- Delay

### MIDI:
- Modwheel
- Pitchbend
- 64 voices polyphony
